# Herdr Terminal — Design Contract

- **Theme name:** `herdr`
- **Display name:** Herdr Terminal
- **Description:** Operational terminal — cyan routing, mint readiness, sharp hacker chrome.
- **Source of truth for tokens:** `src/themes/herdr.ts`
- **Bible target:** `docs/ui/bible/theme-herdr-active.html` → `docs/ui/bible/renders/theme-herdr-active.png` (generated by `scripts/gen-bible-theme-herdr.mjs`)
- **Plan:** issue #312 (Plan 073)

## Intent

Herdr Terminal is a disciplined agent-control terminal, matching Dhruv's
Herdr/Ghostty setup (`neon-blue-split-contrast`). It is **not** a generic green
"Matrix" skin. The runtime should read as near-black operational tooling:

- high contrast, sparse neon, readable for long sessions;
- neon reserved for **routing/focus and state** — never body text everywhere;
- sharp 90-degree box chrome and pure-ASCII sigils;
- clearly related to Herdr without copying Cathedral, Amber CRT, or Obsidian.

### Contrast rules

- `foreground`, `foregroundDim`, `accent`, and all five state colours must stay
  ≥ 4.5:1 against `background`, `surface`, `surfaceRecess`, and
  `surfaceLifted`. Enforced by `src/themes/herdr.test.ts`.
- `divider` is exempt: decorative only, never the sole carrier of text or
  state. Active borders use `accent`.
- Lowest checked pair at recon: approval on lifted surface ≈ 4.81:1.

## State-colour meanings

| State | Colour | Meaning |
|---|---|---|
| idle | mint `#4ECCA3` | ready / healthy |
| thinking | cyan `#00E5FF` | active routing / focus |
| tool | gold `#FFD700` | execution / warning |
| approval | hot pink `#FF3366` | interruption / danger |
| learning | pale gold `#F1D77A` | durable write / learned state |

Cyan is the single primary active/focus signal. Mint means ready. Gold means
execution. Pink means stop-and-decide.

## Chrome

Single-cell, width-stable characters only:

- Frame: `┌ ┐ └ ┘ ─ │` (sharp corners, no rounded Cathedral frames)
- Section sigils: `>` context · `#` memory · `@` mcp · `$` session · `%` registry
- `sectionTracked: false` (no letter-spaced headers)
- Rule: `─` · Tab active: `▸` · Tab inactive: `·` · Bullet: `>`

## Working indicator

Eight-frame ASCII packet progression at 110 ms, coloured `states.thinking`:

```
. : + * # % @ >
```

All frames unique, width 1. Reduced-motion behaviour follows the shared
indicator contract — no Herdr special-casing outside the bundle.

## Prohibited drift

- Matrix rain, falling glyphs, or bright green body copy.
- Glow effects, text shadows, RGB gradients, chromatic aberration.
- Layout changes of any kind — Herdr recolours the locked V2 layout only.
- Cathedral ornament (fleur-de-lis bullets, rounded frames, tracked headers).
- Neon as default body text; body copy stays warm off-white `#F5EFE1`.
- Double-width or ambiguous-width glyphs in chrome or indicator.
